## Env vars
- APP_ENV: string
- DATABASE_URI: string
- PORT: string
- SECRETWORDJWT: string
- REDIS_URI: string
- REDIS_DATABASE: string (number in string)
- RABBIT_URI: string
- RABBIT_ON: string (boolean in string)
- EMAIL_JHONDOE: string
- SECOND_DELAY: string
